# Proyecto: Simulador de Rastreo de Equipos (ELO329)

Este repositorio contiene el desarrollo de las cuatro etapas del proyecto para la asignatura Diseño y Programación Orientada a Objetos. El sistema permite modelar el movimiento y la interacción de diversos dispositivos electrónicos en un entorno bidimensional.
